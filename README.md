## Characterization of thermistors and thermal sensors.

In this first folder, we use ARDUINO UNO to characterize three thermal sensor: NTC100K, PTC100 and the LM35. In particular, we found a great correlation for the first two, while the latter was found to have a non-linear relation with the data.
